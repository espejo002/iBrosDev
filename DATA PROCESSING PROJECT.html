What is A Web Design Package

Web design means planning, creation, and updating of websites.

This is a simple, straight-to-the-point definition of web design.
It basically the act/processs of making websites.

Examples of Web Design Packages

  1. Amaya
  2. iWeb
  3. Net Object Fusion
  4. Webplus x4 (now owned by Microsoft)
  5. Microsoft express web 3.0 and Studio.

Components of A Web Design Package

Solid Layout
In life, a good foundation is necessay for basically everything that exists. Without a good foundation, a building cannot stand. The layout of a website is very important as it serves as the foundation for any website.
Effective Typography
This is also equally important as the layout. The typography is responsible for how your user digests your content. The typography tells the user about your skills in grammar and how good you're at typing without any mistakes.
The Right Colour Scheme
Choosing the right colour scheme is important as it sets the mood of your website. However, the colours you choose should not be dependent on your own personal taste.
Appropriate Design elements
This is where your imagination and creativity can go wild. Here you can use different images, buttons, patterns etc. And combine them together in your website.

Terms Used in Web Design Package

Web Server
This is like the heart of the world wide web. It holds the information that is to be accessed.
URL
This stands for Uniform Resource Locator. It is the address of a particular website/webpage on the internet.
Web Page
It is a special document that displays information to the viewer in form of text, sound/audio or images.
HyperLink
It is an element in a wep page that links it to other related web pages.
